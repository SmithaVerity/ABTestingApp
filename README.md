# AB Testing Server Cafe app

Create an ubuntu instance.

Connect the instance using putty and then run the below script
```
git clone https://github.com/SmithaVerity/ABTestingApp.git
cd ABTestingApp/
sh deployServer.sh
```

For updates
```
sh updateApp.sh
```
